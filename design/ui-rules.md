# UI Rules – Non-Negotiable

## Layout
- The book panel must visually dominate the layout.
- Library and chat panels must feel secondary.
- No layout shift while scrolling or reading.

## Typography
- UI uses Inter.
- Reading content uses a serif font.
- Body text must not be smaller than 16px.
- Reading line height must be ≥ 1.6.

## Chat Panel
- Do NOT use chat bubbles.
- Style as assistant notes, not messenger UI.
- AI responses must be visually calmer than user input.

## Colors
- No gradients.
- No bright greens or reds.
- Accent color only for focus, selection, or emphasis.

## Motion
- All animations ≤ 150ms.
- No bounce or elastic easing.
- Focus mode must feel instant.
